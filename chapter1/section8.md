## 1.8 页面标题、页眉和页脚
- 使用`title`关键字增加页面标题
- 使用`header`和`footer`关键字增加页眉和页脚

```
@startuml
header 这是页眉
footer 图 %page% of %lastpage%

title Example title

Alice -> Bob : message 1
Bob -> Alice : message 2
@enduml
```
